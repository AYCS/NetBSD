groongaをインストール
~~~~~~~~~~~~~~~~~~~~~~~~
 pkgsrcからgroongaをインストールします。

::

 # cd /usr/pkgsrc/textproc/groonga
 # make package-install
 # which /usr/pkg/bin/groonga
 /usr/pkg/bin/groonga                ..... /usr/pkg/bin/以下にインストール
